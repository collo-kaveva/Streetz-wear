STREETZ-WEAR


A Luxury Fashion & Sneaker Boutique


A modern frontend-only ecommerce website built with React.js, Vite, and Tailwind CSS. The project showcases a premium online fashion boutique featuring clothing, sneakers, and accessories from popular brands in a clean, responsive, and visually appealing interface.


Features
Responsive, mobile-first design
Modern homepage with featured collections
Multi-page navigation using React Router
Product catalog with categories and filters
Dedicated pages for sneakers, clothing, and accessories
Product detail pages
Shopping cart and wishlist (frontend state)
Order Goods request page
About and Contact pages
Smooth animations using Framer Motion
Dummy product data stored in JSON for easy future backend integration
Tech Stack
React.js
Vite
Tailwind CSS
React Router
Framer Motion
JavaScript (ES6+)


Project Structure


src/
├── assets/          # Images, icons, and static assets

├── components/      # Reusable UI components

├── pages/           # Application pages

├── data/            # Dummy product JSON data

├── hooks/           # Custom React hooks

├── context/         # Cart and wishlist state

├── layouts/         # Shared layouts

├── routes/          # Route configuration

├── utils/           # Helper functions

└── App.jsx


Installation

Clone the repository:

`
git clone <repository-url>
`

Navigate to the project folder:

`
cd project-name
`

Install dependencies:

`
npm install
`

Start the development server:

`
npm run dev
`

Open your browser and visit:

http://localhost:5173


Build for Production

`
npm run build
`

Preview the production build:

`
npm run preview
`
